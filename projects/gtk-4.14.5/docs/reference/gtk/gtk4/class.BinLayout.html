GtkBinLayout is a GtkLayoutManager subclass useful for create “bins” of widgets.

GtkBinLayout will stack each child of a widget on top of each other, using the GtkWidget:hexpand, GtkWidget:vexpand, GtkWidget:halign, and GtkWidget:valign properties of each child to determine where they should be positioned.
