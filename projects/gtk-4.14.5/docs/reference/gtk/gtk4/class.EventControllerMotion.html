GtkEventControllerMotion is an event controller tracking the pointer position.

The event controller offers GtkEventControllerMotion::enter and GtkEventControllerMotion::leave signals, as well as GtkEventControllerMotion:is-pointer and GtkEventControllerMotion:contains-pointer properties which are updated to reflect changes in the pointer position as it moves over the widget.
