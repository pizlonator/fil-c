Description [src]

This function initiates a font selection operation by presenting a dialog to the user for selecting a font family.

The callback will be called when the dialog is dismissed. It should call gtk_font_dialog_choose_family_finish() to obtain the result.

Available since: 4.10

This method completes asynchronously. Use gtk_font_dialog_choose_family_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
