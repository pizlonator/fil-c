Description [src]

Launch a file manager to show the file in its parent directory.

This is only supported native files. It will fail if file is e.g. a http:// uri.

The callback will be called when the operation is completed. It should call gtk_file_launcher_open_containing_folder_finish() to obtain the result.

Available since: 4.10

This method completes asynchronously. Use gtk_file_launcher_open_containing_folder_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
