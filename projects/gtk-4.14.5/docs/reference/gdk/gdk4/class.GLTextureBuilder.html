GdkGLTextureBuilder is a builder used to construct GdkTexture objects from GL textures.

The operation is quite simple: Create a texture builder, set all the necessary properties - keep in mind that the properties GdkGLTextureBuilder:context, GdkGLTextureBuilder:id, GdkGLTextureBuilder:width, and GdkGLTextureBuilder:height are mandatory - and then call gdk_gl_texture_builder_build() to create the new texture.

GdkGLTextureBuilder can be used for quick one-shot construction of textures as well as kept around and reused to construct multiple textures.
